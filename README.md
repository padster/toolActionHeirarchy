# Tool Action Hierarchy Research

Research for whether action-based or domain-based hierarchies are better for organizing LLM tool calling.

## Overview

This repository contains a Jupyter notebook that explores different approaches to organizing tools when calling Large Language Models (LLMs), comparing domain-based and action-based hierarchies.

## Setup

### 1. Create and activate a virtual environment

On Windows:
```powershell
python -m venv venv
.\venv\Scripts\activate
```

On macOS/Linux:
```bash
python3 -m venv venv
source venv/bin/activate
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Launch Jupyter

```bash
jupyter notebook
```

### 4. Open the notebook

Open `tool_hierarchy_optimization.ipynb` in the Jupyter interface.
